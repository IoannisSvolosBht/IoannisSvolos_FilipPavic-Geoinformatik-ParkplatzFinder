# Lauffähige Location Based App, die einem Nutzer helfen soll, seinen Parkplatz zu finden. 
Entwickelt von Ioannis Svolos und Filip Pavic

![](https://img.shields.io/static/v1?label=&message=Geoinformatik&color=green)
![](https://img.shields.io/static/v1?label=&message=Wintersemeser22/23&color=green)
![](https://img.shields.io/static/v1?label=&message=BHT&color=lightgray)



Unsere App erfordert Java 8, eine mind. SDK 21, compile SDK 32, gradle version 7.2, gradle plugin version 7.1.0, einen Google Maps API key und weitere Abhängigkeiten (dependencies) die in der App in build gradle definiert sind.

# Beschreibung mit Screenshots

* Ein Endbenutzer kann mit der Interaktion mit der Anwendung beginnen, indem er mit einem langen Klick auf die Karte seinen Parkplatz speichert und auf der Karte festlegt. Über dem Marker erscheinen die genauen Koordinaten auf. Dieser Marker hat die genaue Position des Parkplatzes. Er wird auch beim Schließen und erneuten Öffnen der App beibehalten und verschwindet nicht.
